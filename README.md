SkaeleSolutionTemplate
======================

Template for default .NET solution used for Skaele applications

#Usage
- Clone repository and rename the solution file.
- Remove .git folder and create a new git repository. (create repository on github if you feel the need to share)
- Create Environment.config and NLog.config in LocalConfig folder. Examples are in the same folder, just copy paste.
- Create database and update connectionstring in environment.config and nlog.config
- Run admin console application to create database schema and/or create demo data

Kickstart application!
